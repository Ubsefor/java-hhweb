## Вариант 7 (Кадровое агентство) 

*****

Предполагается, что разработанное веб-приложение будет использоваться как система сопровождения работы кадрового агенства.

Функционал приложения обеспечивает систему управления информацией о вакансиях и резюме.

Поддерживаемые типы данных: 

- Люди

  - Фамилия
  - Имя
  - Отчество
  - Образование
  - Домашний адрес
  - История работы: компании, должности (обязанности), зарплаты
  - Статус: ищет работу или нет, если ищет, какие условия (должность, зарплата)

- Компании

  - Название
  - Вакансии: должность + предлагаемая зарплата + требования к образованию и послужному списку

*****
### Список страниц и их описание

1. Страница входа
Содержит: 
	- Поле "Логин"
	- Поле "Пароль"
	- Поле "Вход/Регистрация"
	- Чекбокс "Новый пользователь"
	- Чекбокс "Соискатель/Работодатель", доступный при выборе чекбокса "Новый пользователь"
	- Кнопка "Войти" или кнопка "Зарегистрироваться" при выборе чекбокса "Новый пользователь"

2. Страница регистрации работодателя
Содержит:
	- Поле "Имя компании"
	- Поле "Краткое имя компании"
	- Кнопка "Продолжить"
	- Кнопка "Отменить"

3. Страница регистрации работника
Содержит:
	- Поле "Имя"
	- Поле "Фамилия"
	- Поле "Отчество" – поле не обязательно к заполнению
	- Выпадающий список "Образование" – содержит доступные варианты для графы "Образование" – от "Не указано" до "Доктор наук"
	- Поле "Домашний адрес" – не обязательно к заполнению
	- Кнопка "Продолжить"
	- Кнопка "Отменить"

3. Страница личного кабинета компании
Содержит:
	- Поле "Краткое имя компании"
	- Поле "Полное имя компании"
	- Кнопки "Изменить" около предыдущих двух полей
	- Кнопка "Выйти"
	- Кнопка "Удалить аккаунт"
	- Список вакансий компании
	- Кнопка "Добавить/Редактировать" в списке вакансий
	- Кнопка "Удалить" в списке вакансий
	- Кнопка "Поиск резюме"

4. Страница личного кабинета работника
Содержит:
	- Поле "Имя"
	- Поле "Фамилия"
	- Поле "Отчество" – поле не обязательно к заполнению
	- Выпадающий список "Образование"
	- Поле "Домашний адрес" – поле не обязательно к заполнению
	- Кнопки "Изменить" около предыдущих пяти полей
	- Кнопка "Выйти"
	- Кнопка "Удалить аккаунт"
	- Список предыдущих мест работ
	- Кнопка "Добавить/Редактировать" в списке работ
	- Кнопка "Удалить" в списке работ
	- Список составленных резюме
	- Кнопка "Добавить/Редактировать" в списке резюме
	- Кнопка "Удалить" в списке резюме
	- Кнопка "Поиск вакансий"

5. Страница добавления/редактирования вакансии
Содержит:
	- Поле "Должность"
	- Поле "Зарплата" – поле не обязательно к заполнению
	- Выпадающий список "Образование"
	- Поле "Опыт работы" – поле не обязательно к заполнению, ≥0
	- Кнопка "Сохранить"
	- Кнопка "Отменить"
	- Кнопка "Выйти"
	- Кнопка "Личный кабинет"

6. Страница добавления/редактирования мест работ
Содержит:
	- Поле "Имя компании"
	- Поле "Должность"
	- Поле "Обязанности" – поле не обязательно к заполнению
	- Поле "Дата начала работы"
	- Поле "Дата конца работы" – поле не обязательно к заполнению, в этом случае подразумевается что человек работает в этом месте по настоящее время
	- Кнопка "Сохранить"
	- Кнопка "Отменить"
	- Кнопка "Выйти"
	- Кнопка "Личный кабинет"

7. Страница добавления/редактирования резюме
Содержит:
	- Поле "Желаемая должность"
	- Поле "Зарплата" – поле не обязательно к заполнению, в этом случае подразумевается договорная заработная плата
	- Поле "Контактный номер телефона"
	- Поле "Статус работы" – ищет ли человек в данный момент работу или нет
	- Список мест работ, позволяющий отметить чекбоксом предыдущие места работы, которые необходимо добавить к этой вакансии
	- Кнопка "Сохранить"
	- Кнопка "Отменить"
	- Кнопка "Выйти"
	- Кнопка "Личный кабинет"

8. Страница поиска резюме (для компаний)
Содержит:
	- Поле "Образование"
	- Поле "Предыдущие места работы"
	- Поле "Предыдущие должности"
	- Поле "Предыдущая зарплата"
	- Чекбоксы рядом с предыдущими четырьмя полями, которые позволяют вносить изменения в эти поля и активируют их при поиске
	- Кнопка "Поиск"
	- Кнопка "Выйти"
	- Кнопка "Личный кабинет"
	- Список резюме, найденных при поиске
	- Кнопка "Детальная информация" напротив каждого резюме

9. Страница поиска вакансий (для работников)
Содержит
	- Поле "Компания"
	- Поле "Должность"
	- Поле "Зарплата"
	- Чекбоксы рядом с предыдущими тремя полями, которые позволяют вносить изменения в эти поля и активируют их при поиске
	- Кнопка "Поиск"
	- Кнопка "Выйти"
	- Кнопка "Личный кабинет"
	- Список вакансий, найденных при поиске
	- Кнопка "Детальная информация" напротив каждой вакансии

11. Страница детальной информации
    Содержит все поля от вакансии, резюме или предыдущем месте работы, эта страница доступна при нажатии кнопки "Детальная информация". Также, аналогично остальным страницам (кроме страницы входа), имеются кнопки "Выйти" и "Личный кабинет". При просмотре резюме такая же кнопка есть у каждого места работы в списке прикрепленных к резюме мест работ.

### Сценарии использования

1. Регистрация работника в системе
   - Пользователь попадает на страницу входа/регистрации
   - Пользователь выбирает чекбокс "Новый пользователь"
   - Пользователь вводит логин и пароль и нажимает кнопку "Регистрация", попадая на страницу регистрации работника
   - Пользователь указывает свои персональные данные в соответствующих полях (Имя, Фамилия, Образование, Адрес…) и нажимает кнопку "Продолжить"
   - Далее пользователь попадает в личный кабинет, регистрацию можно считать завершенной, пользователь нажимает кнопку "Выйти"
2. Регистрация компании в системе
   - Пользователь попадает на страницу входа/регистрации
   - Пользователь выбирает чекбокс "Новый пользователь" и "Работодатель"
   - Пользователь вводит логин и пароль и нажимает кнопку "Регистрация", попадая на страницу регистрации компании
   - Пользователь указывает данные компании в соответствующих полях (Краткое имя компании, Имя компании) и нажимает кнопку "Продолжить"
   - Далее пользователь попадает в личный кабинет, регистрацию можно считать завершенной, пользователь нажимает кнопку "Выйти"
3. Авторизация в системе
   - Пользователь попадает на страницу входа/регистрации
   - Пользователь вводит свои логин и пароль в соответствующие поля, не выбирая "Новый пользователь" и нажимает кнопку "Вход"
   - При успешной авторизации пользователь попадает на страницу личного кабинета работодателя или соискателя, в зависимости от типа его аккаунта
   - При неуспешной авторизации пользователь остается на странице входа/регистрации, выводится сообщение о неуспешной авторизации, поля логина/пароля очищаются
4. Добавление предыдущего места работы (для соискателя)

   - Пользователь входит в свой аккаунт на странице входа (пользователь является соискателем)

   - Попав в личный кабинет, пользователь нажимает на кнопку "Добавить" в списке мест работ, попадая на страницу добавления места работы

   - Пользователь заполняет требуемую информацию о месте работы: поля "Имя компании", "Должность"… 

   - Заполнив все необходимые поля, пользователь нажимает кнопку "Сохранить", попадая после этого обратно в личный кабинет

   - В личном кабинете список предыдущих мест работ должен быть обновлен: в него должна быть добавлена только что созданная запись
5. Редактирование предыдущего места работы (для соискателя)
   - Пользователь входит в свой аккаунт на странице входа (пользователь является соискателем)
   - Попав в личный кабинет, пользователь выбирает одно из мест работ в списке мест работ и нажимает на кнопку "Редактировать", попадая на страницу редактирования места работы
   - Страница редактирования места работы отличается от страницы добавления места работы лишь тем, что поля уже заполнены предыдущими значениями, эти поля можно редактировать
   - Заполнив/отредактировав необходимые поля, пользователь нажимает кнопку "Сохранить", попадая после этого обратно в личный кабинет
   - В личном кабинете список предыдущих мест работ должен быть обновлен: в нем должна быть обновлена только что отредактированная запись
6. Удаление предыдущего места работы (для соискателя)
   - Пользователь входит в свой аккаунт на странице входа (пользователь является соискателем)
   - Попав в личный кабинет, пользователь выбирает одно из мест работ в списке мест работ и нажимает на кнопку "Удалить"
   - Список предыдущих мест работ должен быть обновлен: в нем должна быть удалена только что удаленная запись
7. Добавление резюме
   - Пользователь входит в свой аккаунт на странице входа (пользователь является соискателем)
   - Попав в личный кабинет, пользователь нажимает на кнопку "Добавить" в списке добавленных резюме, попадая на страницу добавления резюме
   - Заполнив все необходимые поля и отметив предыдущие места работы выбором соответствующих чекбоксов, которые пользователь хочет прикрепить к резюме, пользователь нажимает кнопку "Сохранить", попадая после этого обратно в личный кабинет
   - Список резюме должен быть обновлен: в него должно быть добавлено только что созданное резюме
8. Редактирование резюме
   - Пользователь входит в свой аккаунт на странице входа (пользователь является соискателем)
   - Попав в личный кабинет, пользователь выбирает существующее резюме в списке добавленных резюме и нажимает на кнопку "Редактировать", попадая на страницу редактирования резюме
   - Страница редактирования резюме отличается от страницы добавления резюме лишь тем, что поля уже заполнены предыдущими значениями, эти поля можно редактировать
   - Заполнив/отредактировав необходимые поля и отметив/удалив предыдущие места работы выбором соответствующих чекбоксов, которые пользователь хочет прикрепить/убрать из резюме, пользователь нажимает кнопку "Сохранить", попадая после этого обратно в личный кабинет
   - В личном кабинете список резюме должен быть обновлен: в нем должна быть обновлена только что отредактированная запись
9. Удаление резюме
   - Пользователь входит в свой аккаунт на странице входа (пользователь является соискателем)
   - Попав в личный кабинет, пользователь выбирает одно из резюме в списке резюме и нажимает на кнопку "Удалить"
   - Список резюме должен быть обновлен: в нем должна быть удалена только что удаленная запись
10. Поиск вакансий (для соискателя)
    - Пользователь входит в свой аккаунт на странице входа (пользователь является соискателем)
    - Попав в личный кабинет, пользователь нажимает на кнопку "Поиск"
    - Возможно получить список вакансий по компаниям, должностям, зарплатам – за это отвечают соответствующие чекбоксы и поля.
    - Результаты поиска поялвяются в соответствующем списке на странице.
    - Пользователь выбирает заинтересовавшую его вакансию, нажав на кнопку "Детальная информация" рядом с этой вакансией
    - В выбранной вакансии видна более детальная информация: все указанные при создании компанией вакансии поля


11. Редактирование персональных данных о себе
    - Пользователь входит в свой аккаунт на странице входа
    - Попав в личный кабинет, пользователь может изменить указанные при регистрации данные о себе или своей компании, нажав кнопку "Изменить" рядом с соответствующими полями, поменяв их и подтвердив изменение
12. Поиск резюме
    - Пользователь входит в свой аккаунт на странице входа (пользователь является работодателем)
    - Попав в личный кабинет, пользователь нажимает на кнопку "Поиск"
    - Возможно получить список резюме по образованию, компаниям, занимавшимся должностям, зарплатам – за это отвечают соответствующие чекбоксы и поля.
    - Результаты поиска поялвяются в соответствующем списке на странице.
    - Пользователь выбирает заинтересовавшее его резюме, нажав на кнопку "Детальная информация" рядом с ним
    - В выбранном резюме видна более детальная информация: все указанные при создании резюме поля и список прикрепленных к резюме предыдущих мест работ.
    - Для каждого прикрепленного места работы можно просмотреть детальную информацию
13. Удаление аккаунта
    - Пользователь входит в свой аккаунт на странице входа
    - Попав в личный кабинет, пользователь нажимает на кнопку "Удалить аккаунт"
    - После подтверждения удаления, пользователь деавторизуется из системы, попадая на страницу входа, регистрационные данные, использовавшиеся им для предыдущего входа более недействительны 

14. Переход в личный кабинет
    - С любой страницы сайта (кроме страницы входа, регистрации и собственно личного кабинета) пользователь может перейти в свой личный кабинет, нажав кнопку "Личный кабинет". При этом все несохраненные/неподтвержденные изменения не сохраняются.
15. Выход из аккаунта 
    - С любой страницы сайта (кроме страницы входа и страницы регистрации) пользователь может деавторизироваться в системе и перейти на страницу входа, нажав кнопку "Выход". При этом все несохраненные/неподтвержденные изменения не сохраняются.
16. Отмена регистрации
    - Попав на страницу регистрации пользователь может в любой момент отменить ее, нажав кнопку "Отменить". При этом аккаунт не будет создан, логин, использовавшийся при создании аккаунта останется свободным.
